# CS2 External Cheat - Usage Guide

## English

### Requirements
- Run CS2 in **Fullscreen Windowed** mode (not Fullscreen)
- Run the cheat as **Administrator**

### Controls
| Key | Function |
|-----|----------|
| F4 | Toggle menu (hide menu to play) |
| F9 | Exit program |

### Important
- **Press F4 to hide the menu before playing**, otherwise mouse clicks will be blocked by the overlay
- When menu is visible: you can adjust settings
- When menu is hidden: click-through enabled, you can play normally

### Features
- ESP Box: Draw enemy boxes
- ESP Health: Health bar on the left side of the box
- ESP Distance: Show distance to enemies
- ESP Snaplines: Lines pointing to enemies

## 中文

### 要求
- CS2 必须使用**全屏窗口化**模式（不是全屏）
- 以**管理员身份**运行本程序

### 按键
| 按键 | 功能 |
|-----|------|
| F4 | 显示/隐藏菜单（隐藏后才能游玩） |
| F9 | 退出程序 |

### 重要提示
- **游玩前请按 F4 隐藏菜单**，否则鼠标点击会被覆盖层拦截
- 菜单显示时：可以调整设置
- 菜单隐藏时：点击穿透，可以正常游玩

### 功能
- ESP 方框：显示敌人方框
- ESP 血量：方框左侧的血条
- ESP 距离：显示与敌人的距离
- ESP 射线：指向敌人的线条
